✂️ Separador JSON por Prestador

Divide archivos JSON grandes en archivos individuales para cada prestador de servicios

💡 ¿Qué hace este módulo?

Este módulo toma un archivo JSON que contiene información de múltiples prestadores (IPS, clínicas, hospitales) y lo divide en archivos separados, uno para cada prestador. Es útil cuando necesitas enviar a cada prestador únicamente sus propios datos.

¿Cómo funciona?

  • Identifica prestadores: Lee el campo "codPrestador" de cada servicio para agruparlos
  • Separa usuarios y servicios: Crea archivos independientes con solo los usuarios y servicios de cada prestador
  • Mantiene estructura completa: Cada archivo generado tiene la estructura JSON completa y válida
  • Genera estadísticas: Muestra una tabla con el número de usuarios y servicios por prestador
  • Empaqueta en ZIP: Descarga todos los archivos JSON en un solo archivo comprimido

📌 Ejemplo de uso: Si tienes un JSON con 17 prestadores y 10,000 servicios, obtendrás 17 archivos JSON individuales, cada uno con solo la información de ese prestador específico.
