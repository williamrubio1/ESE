📊 Conversor JSON → Excel

Visualización rápida de archivos JSON en formato Excel sin transformaciones

💡 ¿Qué hace este módulo?

Este conversor toma uno o varios archivos JSON y los convierte directamente a Excel para visualización. NO aplica correcciones, validaciones ni sustituciones - solo muestra la información tal como está en el JSON.

¿Cuándo usar este conversor?

  • Revisión rápida: Cuando necesitas ver el contenido de un JSON de forma legible
  • Múltiples archivos: Puedes cargar varios JSON y consolidarlos en un solo Excel
  • Sin modificaciones: Ideal cuando solo quieres visualizar sin aplicar reglas de negocio
  • Comparación de datos: Útil para comparar información entre diferentes JSON

Características:

  • Detección automática: Identifica si el JSON es de PYP, BC o Compuestos
  • Soporte múltiple: Carga varios JSON y consolídalos en un solo Excel
  • Hojas separadas: Cada tipo de servicio en su propia hoja (Usuarios, Consultas, Procedimientos, etc.)
  • Sin transformaciones: Los datos se muestran exactamente como están en el JSON
  • Columna de origen: Identifica de qué archivo proviene cada registro

⚠️ Importante: Este módulo NO aplica las validaciones de PYP, BC o Compuestos. Si necesitas procesamiento completo con correcciones y validaciones, usa los módulos correspondientes.

Características del Conversor

  • Detección Automática: Identifica el tipo de JSON (PYP, BC, Compuestos) según la estructura de servicios

📋 Hojas generadas según tipo

PYP:

  • Usuarios, Consultas, Procedimientos

BC:

  • Usuarios, Consultas, Procedimientos, Medicamentos, Otros Servicios, Urgencias, Hospitalización

Compuestos:

  • Usuarios, Consultas, Procedimientos, Medicamentos, Otros Servicios, Urgencias, Hospitalización, Recién Nacidos

⚠️ Diferencias con otros módulos

Este módulo es diferente a PYP/BC/Compuestos:

  • NO corrige fechas según mes evaluado
  • NO completa documentos de profesionales
  • NO aplica sustituciones de diagnósticos o finalidades
  • NO valida códigos CUPS, CIE-10, ni otros estándares
  • NO genera reportes de cambios o validaciones

Usa este módulo solo para visualización. Para procesamiento completo, usa PYP, BC o Compuestos.
