🔄 Conversor Excel → JSON

Conversión rápida de archivos Excel a formato JSON sin validación exhaustiva

💡 ¿Qué hace este módulo?

Este conversor toma un archivo Excel (que ya tenga la estructura correcta de PYP, BC o Compuestos) y lo convierte directamente a JSON. A diferencia de los otros módulos, NO aplica validaciones exhaustivas ni genera análisis detallados.

¿Cuándo usar este conversor?

  • Ya tienes un Excel validado: Cuando has trabajado en un Excel y solo necesitas regenerar el JSON
  • Necesitas conversión rápida: No requieres validaciones complejas ni análisis detallado
  • Pruebas rápidas: Ideal para hacer pruebas de estructura y formato
  • Re-exportación: Cuando quieres volver a generar el JSON después de ediciones menores

Características:

  • Detección automática: Identifica si el Excel es de PYP, BC o Compuestos analizando los nombres de las hojas
  • Tipos de datos correctos: Respeta los formatos definidos (números como números, textos como textos)
  • Descarga inmediata: Genera y descarga el JSON en segundos
  • Sin análisis completo: No genera Excel de salida ni aplica todas las validaciones

⚠️ Importante: Si necesitas validaciones completas, análisis detallado y generación de Excel, usa los módulos PYP, BC o Compuestos en lugar de este conversor.
