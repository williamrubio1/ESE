🔍 OCR Historia Clínica

Extracción de texto de historias clínicas en formato PDF

📋 ¿Cómo funciona?

  1. Selecciona uno o varios archivos PDF de historias clínicas
  2. El sistema extrae automáticamente el texto de cada PDF
  3. Identifica el identificador completo de historia clínica (ej: RC 101_______ -- Jo___ Ca____ Ro___ Lo____)
  4. Genera un archivo .txt por cada HC con ese nombre como archivo
  5. Descarga todos los archivos en un ZIP

✨ Características:

  • ✓ Procesamiento múltiple: Carga varios PDFs a la vez
  • ✓ Limpieza automática: Elimina encabezados y datos estructurados iniciales
  • ✓ Nombres inteligentes: Archivos nombrados según identificador completo de HC
  • ✓ OCR avanzado: Utiliza PyMuPDF para extracción precisa
